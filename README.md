# Senda

> *La senda de tus agentes / The path your agents take.*

**Senda** is a single-user, local desktop manager for AI CLI custom agents and
skills. It's CLI-agnostic by design: you author agents once in a canonical
format and Senda transpiles them to whichever CLIs you target — GitHub Copilot
CLI, Claude Code, or Gemini CLI. It also connects opt-in to Git repositories
(GitHub / Azure Repos / generic Git) and runs a local scheduler so your
automations work in the background.

It is **not** an orchestrator (no LangGraph, no n8n) and not a replacement for
your CLI subscription. Senda spawns the CLI you already pay for and captures
its output.

> Status: pre-alpha — Phase 0 of the implementation plan.

## Repository layout

```
senda/
├── apps/
│   └── desktop/                  Tauri 2 + SolidJS shell (frontend + backend)
│       ├── src/                  TypeScript / SolidJS UI
│       └── src-tauri/            Rust binary, IPC commands
├── crates/
│   ├── core/                     Shared domain types (Agent, Repo, Automation…)
│   ├── acp-client/               Async ACP (Agent Client Protocol) client — publishable
│   ├── git-providers/            RepoProvider trait + GitHub/Azure/Generic impls
│   ├── agent-parser/             Canonical-format parser + per-CLI transpilers
│   └── scheduler/                Cron / event / webhook / manual scheduler
├── packages/
│   └── shared-types/             TS types — hand-written today, generated by tauri-specta in Phase 1
├── docs/
│   ├── architecture.md
│   └── format-conversion.md
└── .github/workflows/ci.yml      Rust + TS CI on every push
```

## Quickstart

```bash
# Install Rust (https://rustup.rs/) and pnpm 10+, Node 20+.
pnpm install
pnpm dev          # Tauri dev mode — opens the window with hot reload
```

### Useful commands

```bash
# Rust
cargo build --workspace
cargo test  --workspace
cargo clippy --workspace --all-targets -- -D warnings

# Frontend
pnpm -r typecheck

# Live ACP integration test (needs an ACP-capable agent in $PATH)
SENDA_ACP_BIN=claude-code-acp cargo test -p senda-acp-client -- --nocapture
```
